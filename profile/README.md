高效的局域网互联互通工具，好看的设计才好用。

像聊天一样，熟悉易上手的传输操作。
- 熟悉的聊天交互。聊天框和功能入口，都以你最熟悉的方式呈现。
- 无需登录。在基础场景，Flix 无需登录即可立即使用。

免流量疾速快传，各种设备都能用。
- 多种方式连接。有线或无线、WiFi 或热点都能连，网页版也已准备就绪。
- 多端同步支持，跨品牌无缝传输。全面支持 Windows，macOS，iOS，Android，Linux。
- 精心优化传输体验。Flix 支持后台传输、断点续传，让传输过程更流畅。

全能的局域网工具，现在更全能。
- 投屏和远程控制。局域网内高效协同，边聊边看边操作。
- 远程文件。远程访问并操作其他设备的文件。
- 副屏扩展。电脑屏幕一键扩展到其他设备，多屏协同更高效。
- 音视频通话。局域网内通话，无需连接互联网，更稳定，更安全。
- 文件云备份。传输的文件可一键备份到云端账号长期保存。
- 跨设备复制粘贴。一台设备复制，另一台设备可直接粘贴。
- 通知流转。多设备通知共享，及时同步最新消息。
